# 📘 CoreSuite — Usage Guidelines

## Logos
- Utiliser `logo.svg` pour les pages principales.
- Utiliser `logo-mono.svg` pour les contextes sombres ou très petits.
- Utiliser `logo-mark.svg` pour les favicons, avatars, badges.

## Couleurs
- Toujours utiliser la palette officielle.
- Le gradient principal doit être utilisé pour les éléments clés.

## Espacement
- Laisser un padding minimal de 16px autour du logo.

## Interdits
- Ne pas déformer le logo.
- Ne pas changer les couleurs du gradient.
- Ne pas ajouter d’effets supplémentaires.
