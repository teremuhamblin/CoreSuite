# 🚀 CoreSuite v1.0 — Release Notes

Version initiale stable et minimaliste de CoreSuite.  
Cette version fournit une base propre, modulaire et extensible pour les futures évolutions.

---

## 🎉 Nouveautés principales
- Module `core.js` (analyse basique + résumé)
- CLI minimaliste (`cli.js`)
- API Express (`api.js`)
- Module `analyzers/basic.js`
- Module `generators/summary.js`
- Documentation initiale (`overview.md`, `roadmap.md`)
- Workflow GitHub CI minimal
- Intégration CodeQL (analyse de sécurité)
- Changelog v1.0
- Milestone v1.0 complétée
- Issue v1.0 stable complétée

---

## 📦 Objectif de cette version
Fournir une version **simple**, **stable**, **fonctionnelle**, et **prête à étendre** pour les prochaines releases (v1.1, v1.2, v2.0).

---

## 🛠️ Compatibilité
- Node.js 18+ recommandé  
- Fonctionne sur Linux, macOS, Windows

Voir `docs/roadmap.md` pour les versions futures.
